The more I learn HTML, the more I realize how many things were created just to simplify the code. Sadly, most programmers haven’t even heard of them, which makes them kind of useless. I’d like to share some tips with you, because using simple lines of code can save a lot of your time.

onclick

onclick=”” is an HTML attribute that causes an element to execute JS code when it is clicked. It means you don’t have to type smth.addEventListener(“click”, () => {}); (or smth. on(“click”, () => {}); if you use JQuery) each time. The fact that this is so little known kills me, because you can make your code much clearer with it. Here’s an example of its usage:

<p>Hi!</p> 
Hi!

See? Less code! And it goes further! If your elements share the same function with different variables, you can just name them in the onclick attribute!
